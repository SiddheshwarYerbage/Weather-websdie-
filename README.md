# Weather-webside
This Weather App is a responsive web application built using HTML, CSS, and JavaScript that allows users to search for real-time weather information by entering a city name. It integrates with the OpenWeatherMap API to fetch current temperature, weather conditions, and dynamic weather icons


💻 How It Works

User enters a city name.
App fetches:
Current weather data
Temperature
Weather condition icon
Using city coordinates, it fetches:
Air Quality Index (AQI)
Main pollutant
Data is displayed dynamically on the UI.
